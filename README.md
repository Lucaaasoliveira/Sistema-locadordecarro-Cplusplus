# Sistema-locadordecarro-Cplusplus
Este projeto é um sistema simples de gerenciamento de locadora de veículos, desenvolvido em C++, com armazenamento de dados em arquivos.
O sistema permite:
Cadastrar veículos na frota
Realizar locações de veículos
Registrar devoluções
Calcular multa por atraso
Gerar relatório de devoluções do dia
